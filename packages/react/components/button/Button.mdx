### Usage
Buttons can be created as different HTML elements such as `a`, `button`, or `input`. Set the `markup` prop to define the type of element.

You can customize the button with various props like `iconName`, `children`, `markup`, `disabled`, `loading`, `fullwidth`, `variant`, `id`, and `className`.
